### 漏洞发现

根据8088端口上传的文件名称`katana_2.php` 编写小脚本 

```bash
import os

ip = '10.0.2.21'
ports = ('80','7080','8088','8715')
file = 'katana_2.php'
for port in ports:
    url ='http://'+ ip +':'+port +'/'+ file
    print(url)
    os.system('curl -I %s ' % url)
```
